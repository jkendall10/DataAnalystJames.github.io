The data journey...

The social media data set contains responses to a series of questions by young adults from across the world.
As you will see, it contains interesting data on the how long people spend on social media and the impact on their
lives.
After importing the data into SQL Server and correcting data types, I assigned an index as student_id to to improve usability. I then began the the process of
cleaning the data before exporting it to Tableau for creating visualisations.

Two columns contained float data which required rounding to make the data more managable. As you can see the the images below
I updated the data set using the UPDATE and ROUND functions.
Following that, a column which contained binary data (1,0) as a response to if socail media had affected academic performance.
As shown below, I changed the type of this column and changed the binary response to 'yes' and 'no' to make it easy to manipulate
when presenting visualisations.

After cleaning the data, I exported it from SQL Server into Tableau to create visualisations. As you can see from the images below and the
dashboard link above, I created a number of visuals to explain the story of the data.
The bar chart below shows top 10 countries by average use of social media in hours and demonstrates that there are six countries across
the world where young people use social media for over six hours a day.

This led me to ask a number of other questions, including what negative impacts does social media have. The dashboard shown below can
be accessed on the link above and reveal some concerning results, including that 18 year olds have had on average of over 3.5 conflicts on
social media, the most of any age group.
