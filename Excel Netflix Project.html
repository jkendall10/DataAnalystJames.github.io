The data journey...

The data set is over 8000 film and tv shows from Netflix up to 2021. It required a number of process to clean it before it could be used,
below are the steps taken to clean the data before using pivot tables to create visualisations. The link above contains the workbook with
both the raw and cleaned data, as well as the created visuals.
1. Set date to correct format using =TEXT and set as date type.
2. Used =TEXT to standardise format of ID so can be sorted and filtered on an index.
3. Checked and removed duplicates.
4. Used =REGEXREPLACE to remove special characters from directors and add space.
5. Used =IF to add comma to end of lone categories and then =LEFT to remove additional categories on the first comma.
6. Used =XLOOKUP to split duration column into separate film and show duration as different formats.
7. Used =replace to remove mins and TRIM to remove spaces from duration and converted to integer.
