The data journey...

The data set is over 1000 customers data from a bike shop. The data needed cleaning and transforming before it could be used to create
pivot tables and an interactive dashboard to highlight age, salary and gender demographics, and income and commuter data. The original
data set (which can be accessed on the button above) contained initials for gender and marital status, which needed transforming to
improve usability. The outcome can be seen below.

I then moved on to cleaning the income data, changing the data type to 'number' to make it easier to work with when using pivot tables.
I also removed the '$' using the REGEXREPLACE function, which was vital to be able to work with the data correctly.

I then moved on to creating an additional column with age data in. Due to the wide range of ages, gaining useful data from ages was not
possible. If used a series of IF functions to create bins of age ranges so the distribution can be more easily seen. As can be seen from the
dashboard, the distribution of age data is clearly seen.

I then create pivot tables looking at customer average income, commute and age. I put these pivot tables into an interactive dashboard, which
includes the region of the customer and their marital status. This allows for a deeper understanding of the customer base and provide
direction for future sales strategies.
